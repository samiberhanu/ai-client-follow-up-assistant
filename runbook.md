# Runbook

## Purpose

The AI Client Follow-Up Assistant is designed to help small business staff organize client notes and generate professional follow-up messages. The workflow is intended to improve consistency, communication, and documentation.

---

# When to Use

Use this workflow when:

* Client notes are messy or incomplete.
* Documents are missing.
* Appointment reminders are needed.
* Staff need to summarize information before contacting a client.
* Internal notes need to be organized.

---

# Inputs

Examples of information that can be provided:

* Phone call notes
* Appointment details
* Lists of missing documents
* Questions from clients
* Expense summaries

---

# Process

### Step 1: Gather Notes

Collect all available information from emails, calls, or internal records.

---

### Step 2: Provide Notes to Claude

Paste the notes into the AI Client Follow-Up Assistant prompt.

---

### Step 3: Review Output

Verify:

* Accuracy
* Professional tone
* Missing information
* Clear next steps

Human review should always occur before sending a message.

---

### Step 4: Revise if Necessary

If the response is unclear:

* Ask Claude to simplify the message.
* Request additional clarification.
* Remove unnecessary details.
* Verify assumptions.

---

### Step 5: Send Final Message

After review, send the message to the client.

---

# Limitations

This workflow:

* Does not replace human judgment.
* Should not make assumptions.
* Should not be used as a substitute for professional advice.
* Requires staff review before communicating with clients.

---

# Privacy Guidelines

Avoid including:

* Social Security numbers
* Bank account numbers
* Tax identification numbers
* Passwords
* Sensitive personal information

Use fictional or redacted examples whenever possible.

---

# Handoff Instructions

A new employee should be able to:

1. Read the prompt.
2. Follow the examples.
3. Review outputs using the evaluation checklist.
4. Revise responses when necessary.
5. Continue using the workflow without additional training.

---

# Future Improvements

Potential enhancements include:

* Spreadsheet integration.
* Document tracking.
* Workflow automation.
* Additional evaluation criteria.
* Support for more types of client requests.
