# AI Client Follow-Up Assistant

A Claude-based workflow that transforms messy client notes into organized action items and professional follow-up messages for small business operations.

---

# Overview

Small businesses often deal with incomplete notes, missing documents, and inconsistent client communication. This project demonstrates how AI can support administrative workflows while maintaining human oversight.

---

# Problem

Client information frequently arrives in different formats, making it difficult to:

* Identify missing documents
* Maintain consistent communication
* Organize internal notes
* Provide clear next steps

---

# Solution

This workflow uses Claude to:

* Organize client notes
* Identify missing information
* Generate professional follow-up messages
* Create internal summaries
* Support quality control through human review
